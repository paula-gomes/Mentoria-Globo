# Mentoria-Globo

# Como rodar a aplicação

### Clone o projeto
```
$ git clone https://github.com/paula-gomes/Mentoria-Globo.git
```
### Instale as aplicações
 ```
 npm i express --save
 npm i nodemon --save-dev
 
```
### Execute a aplicação
 npm start
